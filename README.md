Praktikum ini mengimplementasikan konsep Text-to-Image Generation berbasis Transformer dengan memanfaatkan mekanisme cross-attention antara representasi teks dan token visual menggunakan dataset pokemon_370. Caption teks dibentuk dari nama file gambar dan diproses melalui TextVectorization. Token visual disimulasikan secara acak sesuai modul praktikum, sehingga hasil inference berupa noise, yang merupakan kondisi normal karena tidak diterapkannya decoder visual (VQ-VAE). Fokus utama praktikum ini adalah memahami alur dan konsep dasar arsitektur Text-to-Image, bukan kualitas visual hasil keluaran.
